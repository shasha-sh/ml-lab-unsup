Описание
--------

Лабораторная работа по обработке текстов на примере датасета SMS (классификация спама).
В ноутбуке сравниваются подходы выбора признаков и их влияние на качество классификации и кластеризации.

Проделанная работа включает
----------------------

- Подготовку данных
- Выбор признаков с помощью собственных реализаций и библиотечных методов chi2
- Оценка качества классификации
- Кластеризация с визуализацией
